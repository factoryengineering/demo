# Festify

An event ticketing and show promotion management API built with ASP.NET Core and Entity Framework Core.

## Prerequisites

- [.NET 10 SDK](https://dotnet.microsoft.com/download/dotnet/10.0)

## Build

```bash
dotnet build
```

## Run

```bash
dotnet run --project ./Festify.Api
```

The API starts on **http://localhost:5114** by default.

### API Reference

Once the application is running, open the Scalar API Reference page to browse and test endpoints interactively:

**http://localhost:5114/scalar/v1**

The raw OpenAPI document is available at:

**http://localhost:5114/openapi/v1.json**

## Test

```bash
dotnet test
```
